## Using

Follow the [install guide for Typescript, Webpack and Vue here](https://github.com/HR-CMGT/vue-workshop/blob/master/presentation/install.md).

```
git clone https://github.com/HR-CMGT/vue-workshop-finished.git
npm install
code .
webpack --watch
```
